## Python, C++ and Java

### for-loops

Goal: print `0` to `4` in the terminal.

Python 

```python
for i in range(5):
	print(i)
```

C++
```c
for (int i = 0; i < 5; i++) {
	cout << i << "\n";
}
```

Java
```java
for (int i = 0; i < 5; i++){
	System.out.println(i);
}
```
