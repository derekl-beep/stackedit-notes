## Introduction 

### Programming Paradigms

Two most popular programming paradigms these days:
- Functional
- Object-oriented 

Other paradigms incluede:

- Procedural
- Event-driven
- Logic
- Aspect-oriented

### Benefits of OOP

- Reduce complexity
- Easier maintance
- Code reuse $\rightarrow$ Faster development
