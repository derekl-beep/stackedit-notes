
## Java Project Structure

Project folder
`src`: source file folder
`com.xxxx`: package folder
`Main.java`: main file


## Linked List

### Properties

A linked list

- consists of a group of nodes in sequence
 - grows and shrinks dynamically
 - is made up of two parts: data and reference to the next node
 - `Head`: first node
 - `Tail`: last node
 
 ### Time Complexity

Look up

- By value: $\text{O}(n)$
- By index: $\text{O}(n)$
